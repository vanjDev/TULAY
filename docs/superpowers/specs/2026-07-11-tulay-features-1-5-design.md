# Project T.U.L.A.Y. — Features 1–5 (Approach A)

**Date:** 2026-07-11  
**Status:** Implemented on website

## Decision

Public campaign **Features 1–5** follow the **core idea**, not event logistics alone.

**Progression:** Tolerance → Acceptance → Inclusion → Belonging  
**Tagline:** Bridge to Belonging — Building Connections Beyond Labels

## Features 1–5

1. **See the Gap** (Tolerance ≠ Inclusion) — name the problem  
2. **Name the Quiet Harm** (Microaggressions) — why the gap stays open  
3. **Hear Each Other** (Stories & Bridge Circles) — connection online + in person  
4. **Practice Belonging** (Guided reflection) — quiz + circle themes  
5. **Leave Your Mark** (Commitment & belonging) — physical planks + digital pledge  

## Bridge to Belonging (live activity)

Flagship campus activity implementing Features 3–5. Day-of steps (registration → grouping → interaction → prompts → planks → closing) live under `/bridge`, not as the top-level feature list.
